# Bihar Streets

A browser-playable, original Bihar-inspired open-world action prototype. It offers a fictional Patna-style map with ghats, markets, station roads, fairgrounds, village fields, an auto-rickshaw, delivery missions, and a full-screen retro action-game HUD and a pseudo-3D third-person chase camera.

This project intentionally avoids copying protected game assets, maps, characters, mission text, branding, or code from any commercial title. It is an original homage to the broader open-world genre.

## Run

Open `index.html` in any modern browser, or serve the folder with a static server:

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

Then visit <http://127.0.0.1:8000>.

## Controls

- `W` / `S` or up/down arrows: walk / reverse
- `A` / `D` or left/right arrows: turn the third-person camera
- `Shift`: sprint / accelerate
- `E`: enter or exit the auto-rickshaw
- `Space`: start or complete missions at glowing markers
- `M`: toggle the mini-map
